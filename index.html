meteor-bootstrap-grayscale

A brochure site using the Grayscale template from Start Bootstrap, converted by Chip Castle Dot Com, Inc. to work with Meteor.

Getting Started

To use this as a brochure site, do the following to get started:

  • Install Meteor using the instructions on their site for your platform

  • Download this repository on GitHub

  • Unzip the archive

  • Change into the directory, type meteor and press the Enter key

  • Visit the site by pointing your browser to http://localhost:3000

  • Edit the index.html and public/css/grayscale.css files to suit your needs.
